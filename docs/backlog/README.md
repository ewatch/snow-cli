# Backlog

This directory tracks planned work items for snow-cli, organized by phase.

## How to Use

Each file represents a work phase. Items within are ordered by priority.
Use the status markers:

- `[ ]` — Not started
- `[x]` — Completed
- `[-]` — Cancelled / deferred

When picking up work, check the current phase file for the next available item.
Complete items in order unless dependencies dictate otherwise.
